in stock products filter widget 
ابزارک فیلتر محصولات موجود
== Description ==
این پلاگین برای شما یک ابزارک ایجاد میکند که مخاطبان سایت بتوانند در فروشگاه فقط محصولات موجود را ببینند
و به صورت انتخابی میشود محصولات ناموجود در انتهای لیست محصولات نمایش داده شوند.
میتوانید به صورت خودکار قیمت را از محصولاتی که موجودی آنها تمام میشود حذف کنید

this plugin add widget to your store for customer that can show only in stock products
and by option, show in stock product first in list of products
Remove price from sold out product page automaticly


== Installation ==
1. افزونه را یا مسقیما آپلود کنید و یا از طریق مخزن وردپرس آنرا نصب نمایید
2. از بخش ابزارکها ابزارک موردنظر را اضافه کنید

1. install from wordress or upload directly
2. in widget area add widget to sidbar
